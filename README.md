This project is a COVID-19 Data Dashboard created using Python, Dash, and Plotly Express.
It visualizes global COVID-19 statistics — including confirmed cases and deaths — in a clean, interactive interface.
Users can filter data by country, update charts dynamically, and gain insights from visual analytics.

Key features:

Real-time data filtering by country or region

Interactive charts built with Plotly

Responsive dashboard layout using Dash components

Easy to extend with additional data sources or KPIs
